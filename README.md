# Fake Stock Simulator

Open `index.html` in your browser to run the website.

Pages:
- `market.html` — view available fake stocks and price changes
- `trade.html` — buy and sell stock shares
- `create.html` — create new fake stocks
- `portfolio.html` — review your holdings and performance
- `settings.html` — reset the simulator and view activity

Features:
- Start with $750 in cash
- Market is open from 05:00 to 22:00 UTC only
- Create fake stocks with custom tickers and names
- Buy and sell shares using cash balance
- Track portfolio value, net worth, and gain/loss
- Simulated live price movement every 5 seconds while the market is open
- Activity feed and localStorage persistence
